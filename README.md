# SmartRotors
![SmartRotors](./Mod/thumb.png)

* [Info](#info)
* [Credits](#credits)

## Info

This mod adds Smart Rotor blocks with special functionality.
Right now there is only the solar module and some generic rotors.

The solar rotors will try to point to the sun.
